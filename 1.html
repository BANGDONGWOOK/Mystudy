HTML이란?

HTML(Hyper Taxt Markup Languege)은 프로그래밍 언어는 아니고, 우리가 보는 웹페이지가 어떻게 구조화되어 있는지 브라우저로 하여금 알 수 있도록 하는 마크업 언어이다. HTML은 elements로 구성되어 있으며, 이들은 적절한 방법으로 나타내고 실행하기 위해 각 컨텐츠의 여러 부분들을 감싸고 마크업한다. tags는 웹 상의 다른 페이지로 이동하게 하는 하이퍼링크 내용들을 생성하거나, 단어를 강조하는 등의 역할을 한다.

HTML 문서의 구조

  1. !DOCTYPE html : 문서의 형식을 표현.
  2. html tag : 전체 페이지의 컨텐츠를 포함하며, 기본 요소.
  3. head tag : 홈페이지 이용자들에게는 보이지 않지만, 검색결과에 노출될 키워드, 홈페이지 설명, CSS 스타일, character setdeclaration 등 HTML 페이지의 모든 내용을 포함.
  4. meta charset="utf-8" : HTML 문서의 문자 인코딩 설정을 UTF-8로 지정. 이를 통해 페이지에 포함된 모든 텍스트 내용을 처리 가능.
  5. title tag : 페이지 제목 설정. 페이지가 책갈피가 될 때 해당 페이지를 설명하는 역할.
  6. body tag : 텍스트, 이미지, 비디오, 게임, 재생 가능한 오디오 트랙 등을 비롯하여 페이지에 표시되는 모든 컨텐츠들을 포함.

HTML의 다양한 태그

  1. "html" : 웹페이지의 시작과 끝.
    "!DOCTYPE"은 문서 유형을 지정하는 것. 웹 브라우저가 이 페이지를 어떻게 해석해야 하는지를 알려준다.
  2. "head" : 웹페이지의 정보, 문서에서 사용할 외부 파일들을 링크할 때 사용
  3. "body" : 브라우저에 실제 표시되는 내용
  4. "title" : 문서 제목. 웹브라우저의 제목 표시줄에 표시된다.
  5. "div" : 아무런 의미가 없으며, 컨텐츠들을 어떤 목적에 따라 묶어야 할 때 사용된다.
  6. "span" : 아무런 의미가 없으며, 컨텐츠들을 어떤 목적에 따라 묶어야 할 때 사용된다.
  7. "a"
    8. : anchor, 웹페이지나 외부 사이트 연결. 링크로 사용할 텍스트나 이미지를 'a'로 묶고 href(hypertaxt reference)속성을 이용해서 연결할 웹페이지의 이름이나 웹사이트 주소를 지정하면 된다.
    href='#'은 실제로 연결되지 않은, 링크역할만 하도록 만든 것으로 널 링크(NULL link)라고 한다.
  8. "script" : 코드 삽입. 실행 가능한 코드를 웹페이지에 포함시키거나 참조하기 위해서 사용한다. 보통 Javascript 코드를 넣을 때 사용한다.
  9. "link" : 외부 파일을 연결할 때 사용한다.
  10. "img" : 이미지 삽입. "img src=이미지 경로"로 작성한다.

    11. image 속성
    width : 너비 heigh : 높이 alt : 대체 텍스트 title : 툴팁 usemap : 이미지 맵

  11. "p" : 단락을 구분할 때 사용한다.
  12. "li" : 항목을 나열할 때 사용한다. ol과 ul의 자식 태그이다.
  13. "ul" : under list, 순서가 없는 list
  14. "ol" : order list, 순서가 있는 list
  15. "style" : 스타일 정보를 정의할 때 사용한다.
  16. "br" : 줄을 바꿀 때 사용하며, 닫는 태그가 없다.
  17. "hn" : heading tag, h1-h6까지 있으며 숫자가 커질수록 텍스트 크기는 작아진다.
  18. "strong" : 중요한 내용을 강조하고 싶을 때 사용한다. 텍스트를 진하게 한다.
  19. "b" : bold, 아무 의미 없이 텍스트를 진하게 하고 싶을 때 사용한다.
  20. "em" : emphasize, 중요한 내용을 강조하고 싶을 떄 사용한다. 텍스트를 기울임꼴로 표시한다.
  21. "i" : italic, 아무 의미 없이 텍스트를 기울임꼴으로 표시하고 싶을 때 사용한다.
  22. "u" : underline, 텍스트에 밑줄을 긋고 싶을 때 사용한다.
